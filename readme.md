# 云微信小程序

## 项目介绍

基于微信小程序开发的网易云音乐小程序，支持登录、搜索、播放、收藏、评论、分享等功能。

- 2023.11.26 首页轮播图区域，无法给两张轮播图之间设置间隔
- 2023.11.26 首页轮播图区域, 无法给轮播图项 swiper_item 设置圆角,就是无法设置圆角裁切

- 2023.11.28 首页推荐歌单方形图片下的文本无法设置超出两行文本显示省略号
